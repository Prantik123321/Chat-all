# Chat-Box - Real-time Group Chat Application

A full-featured real-time chat application built with Flask (Python) and Socket.IO.

## Features

- Real-time messaging with Socket.IO
- User authentication (name-based)
- Multiple chat rooms
- File sharing (images & videos)
- Emoji support
- Online users list
- Light/Dark theme toggle
- Responsive design
- Chat history
- Typing indicators
- Notifications
